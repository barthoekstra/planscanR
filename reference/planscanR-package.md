# planscanR: retrieve environmental-assessment records from European portals

A unified R interface for fetching environmental-assessment records
(EIA, SEA, follow-up advice) from European national portals. See
[`get_assessments()`](https://barthoekstra.github.io/planscanR/reference/get_assessments.md)
for the main entry point and
[`get_assessments_coverage()`](https://barthoekstra.github.io/planscanR/reference/get_assessments_coverage.md)
for the list of supported countries and portals.
